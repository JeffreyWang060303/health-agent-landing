# Health Agent — Landing Page

Frontend repository for **Health Agent**, an AI-powered decision support tool for wellbeing.

## What is Health Agent?

Health Agent delivers a trustworthy health signal at the exact moment you need it—built directly into the apps you already use (e.g. food delivery, grocery). Instead of tracking after the fact, we help you make better choices *before* the decision is made.

## Key Features

- **Real-time intervention** — Surfaces contextual suggestions (e.g. via Dynamic Island) when you're about to choose a meal or product
- **Intelligent insight** — Analyzes ingredients and predicts energy impact automatically, without manual logging
- **Holistic health score** — Turns complex biomarkers into a single, actionable daily signal
- **Contextual guidance** — Chat with the agent to understand the "why" behind your data and get personalized tips

## Tech

- Vanilla HTML, CSS, and JavaScript
- Responsive design with dark mode support
